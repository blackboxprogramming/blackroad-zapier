## Features

- **Triggers**
  - New Deployment Created
  - Deployment Completed
  - Analytics Threshold Reached

- **Actions**
  - Create Deployment
  - Track Event
  - Find Deployment

- **Searches**
  - Find Deployment by ID

2. Create new integration
3. Import this codebase
4. Configure authentication
